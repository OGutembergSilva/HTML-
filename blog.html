Como foi o surgimento do HTML?

Ele foi inventado por Tim Berners-Lee no ano de 1991,Tim era um fisico do centro de pesquisas CERN(European Council Research)na Suiça. Tim Berners teve a ideia de criar um sistema de hipertext hipertexto na internet. Hipertexto significa um texto que possui referências (links) para outros textos que podem ser acessados imediatamente. O HTML veio para facilitar a comunicação e disseminação de documentos entre os pesquisadores. Ele está na versão HTML5.

"A Web como eu imaginei, ainda não vimos isso. O futuro ainda é muito maior que o passado." Tim Berners-Lee

Funcionalidade

A função do HTML é permitir que um navegador leia e traduza marcações de textos e outras mídias para uma página na web,permitindo que o usuário possa visualizar e interagir com essa página.Isso inclui vídeos,formulários,áudio e muito mais. Ao HTML não cabe definir características visuais ou comportamentais,pois isso deve ser feito usando CSS e JavaScript. É recomendado que no documento não contenha atributos como bgcolor ou tags como fonts,pois o objetivo do HTML não é definir aparência visual.

Principais Tags

  1. DOCTYPE:Deve ser a primeira a aparecer em uma página HTML para indicar ao browser qual versão da linguagem usada.
  2. Html:Essa tag delimita o documento,então todas as demais tags da página devem estar nesse espaço.
  3. Meta:Ela pode especificar qual conjunto de caracteres(character set ou charset)será usado para renderizar a página.Ela também pode ser utilizada para outros fins,como o SEO(Serch Engine Optimization).
  4. Title:Define o título da página,aquele que aparece na janela/aba do navegador.
  5. Body:Marca o espaço no qual deve ser contido o conteúdo visual da página.As demais tags que representam texto,botões etc,devem ser adicionadas nesse intervalo.
